### 如何更改语言？

打开 `config.yml` 并更改 `lang` 设置。

```yaml
config_files:
  lang: 'en'
  dictionaries-lang: 'en'
```

## 如何编辑或创建新语言？

**插件**消息的**语言**文件位于以下文件夹中：`plugins/ItemsAdder/lang/`

默认的 ItemsAdder 包的物品/菜单**语言**位于每个 `namespace` 文件夹中的 `dictionaries` 文件夹中。\
例如：`contents/iasurvival/dictionaries/`
