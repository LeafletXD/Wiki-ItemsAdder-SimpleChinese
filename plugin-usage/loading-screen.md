# 🔃 加载界面

## 自定义加载界面

您可以使用此工具创建自定义加载界面: [https://non0reo.github.io/ImgToShader/](https://non0reo.github.io/ImgToShader/)

{% hint style="warning" %}
目前该工具不允许插入自定义 logo，只能更改颜色并隐藏 Mojang logo。
{% endhint %}
